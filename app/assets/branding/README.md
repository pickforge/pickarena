# PickArena Brand Assets

PickArena uses the Pickforge Studio v2 visual system: dark canvas, off-white bracket mark, target motif, leaderboard cue, and ember accent.

## Source

These files are exported from `branding-visual/assets/pickarena/`. Parent studio assets are exported from `branding-visual/assets/pickforge/`.

## Files

- `pickarena-mark-128.svg`, `pickarena-mark-light.svg`, and `pickarena-mark-mono.svg` are the canonical mark variants.
- `pickarena-app-icon.svg` plus `pickarena-app-icon-{32,64,128,256,512,1024}.png` are package/app icon exports.
- `pickarena-favicon.svg`, `pickarena-favicon-{16,32}.png`, and `pickarena-favicon.ico` are browser/favicon exports.
- `pickarena-wordmark.svg`, `pickarena-lockup-horizontal.svg`, and `pickarena-lockup-stacked.svg` are lockups for docs and marketing.
- `pickarena-og-image.png` is the social card export.
- `pickforge_mark.png` and `pickforge_logo.png` are the parent Pickforge Studio mark and wordmark.

## Colors

- Canvas: `#0A0A0B`
- Surface: `#0F0F11`
- Text: `#F2F2F3`
- Muted text: `#9A9AA0`
- Ember: `#FF7A1A`
- Ember dark: `#C2410C`
